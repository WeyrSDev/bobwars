# bobwars
bobwars is a simple rts depicting the struggle of bob vs google plus

###building
in order to build the project properly, make sure you have **SFML\include** and **SFML\lib** in the _root_ directory of the project.

###the game
####controls
of camera:
* W = Up
* A = Left
* S = Down
* D = Right 

of test unit:
* UpArrow = Up
* DownArrow = Down
* LeftArrow = Left
* RightArrow = Right
